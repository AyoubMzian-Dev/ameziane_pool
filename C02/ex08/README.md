# Exercise 08: ft_strlowcase

**Function:** `char *ft_strlowcase(char *str);`

**Description:** Convert every letter in the string to lowercase.

**Key Points:**
- Only converts uppercase letters (A-Z) to lowercase (a-z)
- Other characters remain unchanged
- Modifies the original string
- Returns the same string pointer

**Method:** Add 32 to ASCII value of uppercase letters.
